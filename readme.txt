Comprehensive SEO audit system with AI-powered recommendations using Chrome's built-in AI (Gemini Nano) and traditional server-side checks.

== Description ==

SEO Audit with AI is a modern, premium WordPress plugin designed to help you achieve search engine perfection. It combines traditional SEO heuristics with the power of local AI (Chrome's Gemini Nano) to provide actionable insights without sending your data to third-party servers.

Key Features:
*   **Full SEO Audit**: Analyzes over 50+ on-page SEO factors.
*   **Local AI Analysis**: Uses Chrome's built-in AI for deep content analysis and readability improvements.
*   **Premium Dashboard**: Track your site's SEO health with beautiful visualizations.
*   **Background Rescanning**: Automatically keeps your scores up to date.
*   **Elementor Integration**: Audit your pages directly within the Elementor editor.
*   **CSV Export**: Export your audit history for reporting.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/seo-audit-with-ai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'SEO Audit' menu to view the dashboard.
4. Open any post or page to run a detailed AI audit from the sidebar.

== Frequently Asked Questions ==

= Does this require an API key? =
No! This plugin uses Chrome's built-in AI (Gemini Nano) which runs locally on your machine. Ensure you are using a compatible version of Chrome and have the "Optimization Guide" feature enabled.
